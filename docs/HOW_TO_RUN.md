- yolov8n.pt

# set up virtual environment (optional but recommended)
```bash
python -m venv env
source env/bin/activate  # On Windows use `env\Scripts\activate`
```

# install dependencies
```bash
pip install -r requirements.txt
```


# run the application
```bash
python main.py
```
